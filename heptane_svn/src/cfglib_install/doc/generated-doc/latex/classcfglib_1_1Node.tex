\doxysubsection{Detailed Description}
This abstract class represents a \mbox{\hyperlink{classcfglib_1_1Node}{Node}}. Nodes are basic components of a CFG. Nodes can be of type {\ttfamily BB} or {\ttfamily Call} A {\ttfamily Call} node contains instructions including the call instruction.
